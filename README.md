
# astro starter

astro starter preconfigured with tailwindcss, tailwind-fluid-typography, react, astro-seo and astro-robots-txt


## installation

```bash
yarn install
```
    
## Run Locally

Start the server

```bash
yarn dev
```

## Build

```bash
npm run build
```
